# Akshat-Da-Dhaba-Website
A responsive and vibrant restaurant website for 'Akshat da dhaba'.

Welcome to the official website repository for **Akshat da dhaba**! This project is a front-end web design showcasing a local restaurant, complete with a menu, about section, and contact details.

## Features
* **Attractive UI:** A vibrant and welcoming design for food lovers.
* **Responsive Layout:** Looks great on both desktop and mobile screens.

## Tech Stack
* HTML5
* CSS3
